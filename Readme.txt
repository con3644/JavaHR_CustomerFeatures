Instructions for the Customer Features Assignment Page.

1. Open the JazzHR_CustomerFeatures.html file in any web browser. 

2. Click the "About This Page" button for more information or click the "Assign Groups" button to proceed.

3. The customers have now been assigned groups. Enter a customer and a feature in their respective boxes, then click the "Check Access" button to determine whether the customer has access to the feature.
